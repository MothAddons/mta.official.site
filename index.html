The Command

#1 get a command block. For begginers put in text chat /give @s command_block.
#2 make it a repating type.
#3. Next put /execute if block xyz xyz xyz masked run /particle minecraft:particle_name_here.
#4. The first two xyz's are the same so that should the cordinates of the first item frame the last xyz being the second item frame. Note: Do add a 1 tick or more depending on the particle.
